Let's build a React App from scratch! We are going to build a game called tic tac toe and in the process learn the basics of React, master them and be able to use them in any React project of our choice. Let's GO!

Setting Up Local Development Environment

You will need to have NodeJS installed in your Machine. To check if you have NodeJS installed, type node -v in the Terminal. You can use a code editor of your choice. I use VSCode.

Once you have your development environment ready, in the terminal, we are going to run the command: npx create-react-app tic-tac-toe-react-app (you can call your app whatever you wish, I noticed mine is a bit long) Still in the terminal, navigate to your React application directory by running this command: cd tic-tac-toe-react-app Then start your React app by running: npm start
